
## Based on: https://github.com/nboughton/dotfiles
### Applications 

- File Manager: [ranger](https://github.com/ranger/ranger) and thunar
- Browser: Firefox
- Editors: VSCode and Sublime Text
- Database: DataGrip
- Mail: Thunderbird
- Terminal: [Alacritty](https://github.com/alacritty/alacritty)
- Shell: zsh, [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh) with [powerlevel10k theme](https://github.com/romkatv/powerlevel10k)
- Launcher: [Ulauncher](https://ulauncher.io/) (with some amazing extensions)
- Screenshot: [grim](https://github.com/emersion/grim), [slurp](https://github.com/emersion/slurp), [swappy](https://github.com/jtheoof/swappy)
- Screen record: [grim](https://github.com/emersion/grim), [slurp](https://github.com/emersion/slurp), [wf-recorder](https://github.com/ammen99/wf-recorder)
- Bar: [waybar](https://github.com/Alexays/Waybar)
- Notifications: [mako](https://github.com/emersion/mako), [avizo](https://github.com/misterdanb/avizo), [wob](https://github.com/francma/wob)
- For display management: [wdisplays](https://github.com/cyclopsian/wdisplays) and [kanshi](https://github.com/emersion/kanshi)
- Clipboard Manager: [clipman](https://github.com/yory8/clipman)
- GTK Theme: [https://github.com/nboughton/NordSur](https://github.com/nboughton/NordSur)
- Waybar/Taskbar Icons: [https://github.com/nboughton/DarK-fork](https://github.com/nboughton/DarK-fork)
